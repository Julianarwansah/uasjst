# UAS JST – Kalkulator Jaringan Saraf Tiruan

Kumpulan kalkulator interaktif berbasis web untuk perhitungan jaringan saraf tiruan, dibuat sebagai tugas UAS.

## 📂 Isi

- **`index.html`** – Halaman utama (landing page) yang menampilkan dua kalkulator
- **`lvqkreditbank.html`** – Kalkulator LVQ (Learning Vector Quantization) untuk klasifikasi kelayakan kredit bank
- **`seleksipenerimaanbeasiswa.html`** – Kalkulator Kohonen–Grossberg untuk seleksi penerimaan beasiswa

## 🚀 Cara Menggunakan

1. Buka `index.html` di browser, atau akses langsung URL GitHub Pages
2. Pilih salah satu kalkulator dari halaman utama
3. Masukkan data yang diperlukan
4. Klik **Hitung Sekarang** untuk melihat langkah-langkah perhitungan

## ✨ Fitur

- ✅ **Responsive** – Tampilan optimal di HP, tablet, dan desktop
- ✅ **Interaktif** – Input dapat diubah-ubah untuk eksplorasi
- ✅ **Langkah demi langkah** – Setiap tahapan perhitungan ditampilkan
- ✅ **Tanpa backend** – 100% client-side (HTML + CSS + JavaScript)

## 🛠️ Teknologi

- HTML5
- CSS3 (Grid, Flexbox, Media Queries)
- JavaScript (vanilla, tanpa library)
